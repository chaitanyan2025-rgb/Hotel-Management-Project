# 📚 Book Store Management System

A Python + MySQL project to manage books and staff in a bookstore.

## 🚀 Features
- Add, remove, search books
- Add, remove staff
- Display staff details
- Display available books

## 🛠️ Tech Stack
- **Python 3.8+**
- **MySQL 8.0**
- Runs on Windows/Linux
